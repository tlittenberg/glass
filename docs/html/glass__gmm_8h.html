The Expectation-Maximization (EM) Algorithm.

Execulte one iteration of the EM algorithm to update fit to multivariate gaussian model (could be generalized)

E-step: Compute probability for each sample to belong to each mode

M-step: Recompute mean, covariance, and relative weight of newly weighted samples

Parameters
[in]samplesdata points \(x_i\) and probabilities \(p(x_i|k)\) for each Gaussian \(k\)
[in]NPsize of parameter space
[in]NMODEnumber of modes
[in]NMCMCnumber of samples
[out]logLlog-likelihood of input model
[out]BICBayesian Information Criteria (BIC) for input model
[in,out]modesparameters of each Gaussian including relative weights \(\alpha_{k}\), updated by M-step.
Returns
0 if successful, 1 if singular due to zero weight in one of the modes.
